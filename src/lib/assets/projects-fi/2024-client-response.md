## Turun kaupunki - Response Energy Management Dashboard

### Yhteenveto

Dashboard useiden rakennusten ja niiden laitteiden energiankulutuksen seuraamiseen.
Datan käsittelyä ja analysointia. Toimin projektissa johtavana kehittäjänä.

### Teknologiat

- Typescript
- React
- Next.JS
- PostgreSQL
- Docker
- Carbon Design System
